# Express Typescript Template with TypeORM, Typedi and ClassValidator


### Installation

```bash
pnpm install
```

### Format code
```bash
pnpm format
```

### Lint
```bash
pnpm lint
```

### Example environment file

```env
PORT=4001
NODE_ENV=development
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_NAME=db_name
DB_PASS=password!
```
